# housing_price
Code for the Kaggle Housing Price competition in which my final model places in the top 1%.

Using the following techniques and frameworks:
	- Scikit-Learn
		- Preprocess Pipeline
		- RandForestRegressor
		- RandomSearchCV
		- GridSearchCV
		- Ensemble Learning
	- XGBoost
	- CatBoost
	- Optuna Hyperparameter Tuning
	- Feature Engineering
